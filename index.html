Esse projeto destacará os comandos essenciais mais usados no GIT

Saber git é essencial para poder contirbuir para projetos em equipe

  • git commit -m: Comando para adicionar uma mensagem de commit
  • git add: Comando para adicionar arquivos ao stage
  • git push: Comando para enviar as alterações para o repositório remoto
  • git pull: Comando para baixar as alterações do repositório remoto
  • git branch: Comando para criar uma nova branch
  • git checkout: Comando para mudar de branch
